Getting started

Linux is currently the best supported platform (tested with Ubuntu 20.04 LTS). Windows and MacOS are also supported but you might experience more issues. So far there is no support for ARM-based systems.

  • Download the AppImage, .exe or appropriate executable file for your OS. You might need to change permissions to run the executable. (On Windows, also install the Universal USB Drivers and other potentially drivers needed for your device.)
  • Start the desktop app and follow the instructions. You might need to allow or enable the execution of the software.
